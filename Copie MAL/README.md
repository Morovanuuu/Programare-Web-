# WeebHub

Proiectul structureaza un site de pe care se pot vizualiza anime-uri si manga-uri

## Stack Tehnologic:

- **Backend:** Node.js, Express.js
- **Frontend:** EJS (View Engine), HTML/CSS
- **Stocare:** In-memory (array-uri locale pentru useri și anime-uri)

## Cum se rulează local:

1. Instalează dependențele: `npm install`
2. Pornește serverul: `node app.js`
3. Deschide browserul la adresa: `http://localhost:3000`
